**Brio è un'applicazione per creare animazioni in Inkscape.** Può animare qualsiasi elemento SVG e supporta easing curves, diversi keyframes per ogni proprietà, preview delle animazioni direttamente in Inkscape, e così via. Le animazioni vengono salvate direttamente nel file SVG e possono essere renderizzate come file video o animazioni CSS per essere inseriti in siti.
# Esempi di animazioni di Brio ![](BrioAssets/akademy.mp4) ![](BrioAssets/example2.mp4) # Come funziona Brio
Per usare Brio: 1. Crea un file SVG o aprine uno; seleziona un elemento da animare. ![](BrioAssets/inkscape1.png) ![](BrioAssets/launch.png) 2. Clicca su "Seleziona proprietà da animare" per dire a Brio cosa animare. ![](BrioAssets/properties.png) 3. Modifica l'elemento e aggiungi un nuovo keyframe. ![](BrioAssets/titlebar.png) 4. Fine!
# Cosa supporta Brio
Attualmente Brio supporta le animazioni con cambbi di **posizione, dimensione, transform, blur, path, bordi arrotondati, trasparenza, colore interno e bordo**. Nella prima versione ci sarà il supporto anche per **gradienti, gradient mesh, testo**.
# Per cosa usare Brio
Le animazioni di Brio possono essere renderizzate come video a qualsiasi risoluzione e framerate. La trasparenza dello sfondo è supportata, per cui il video finale può essere montato in sovraimpressione a un altro. Le animazioni possono anche essere esportate come animazioni CSS, il che permette di inserire l'SVG animato direttamente su una pagina web in modo nativo.
